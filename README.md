# Go4Learning


#### 软件构造基础 2024春 小组结课大作业

项目名称：GO4Learning

旨在提高学习计算机的同学的学习效率，提供一站式的学习资源平台以及智能问答的大语言模型。

前后端分离设计，前端主要采用Vue + tauri 框架，后端采用C# + EF + WebAPI框架，数据库使用MySQL。
